Keyboard shortcuts
  • Alt + ← = Previous page
  • Alt + → = Next page
  • Ctrl/⌘ + S = Save
  • Shift + mouse wheel (on image) = Zoom image (min = fit size)
  • Shift + mouse wheel (on text) = Increase/decrease font size
  • Mouse drag on image Pan (only when zoom > 1)
  • Swap = Swap image/text panels
  • ? Toggle this help
  • Esc Close
  • Edit URL hash like #page_stem to jump
Tip: Zoom is centered under the mouse cursor.

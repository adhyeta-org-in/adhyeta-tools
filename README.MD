# Adhyeta Tools

This project provides various tools used for proofreading and other activities.

## llm-proof

Contains three tools.

`extract-pdf` extracts flattened images from a given pdf file

`start-llamacpp` starts a llama-cpp server using provided parameters. Look at the `TUNABLE CONFIG` section to modify stuff that cannot be tuned via the cli

`start-ocr` performs OCR of the extracted images using a model running on llama-cpp. Look at the `TUNABLE CONFIG` section to modify stuff that cannot be tuned via the cli
